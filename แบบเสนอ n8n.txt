ชื่อโครงการ
Smart Money Buddy ระบบผู้ช่วยวางแผนการเงินอัจฉริยะผ่าน LINE Chatbot ด้วยเทคโนโลยี n8n และ AI
1. หลักการและเหตุผล
ในยุคปัจจุบัน ผู้คนส่วนใหญ่มักประสบปัญหาในการจัดการรายรับ-รายจ่าย และขาดการวางแผนการเงินที่เป็นระบบ ส่งผลต่อความมั่นคงทางการเงินในระยะยาว โครงการนี้จึงเสนอแนวทางในการพัฒนา "Smart Money Buddy" ซึ่งเป็นระบบบันทึกรายรับ-รายจ่าย และวางแผนการใช้จ่ายรายเดือน โดยเชื่อมต่อกับ LINE Chatbot ที่ผู้ใช้งานคุ้นเคย พร้อมใช้ เทคโนโลยี n8n ในการจัดการกระบวนการอัตโนมัติ และ AI (เช่น GPT) ในการวิเคราะห์พฤติกรรมการใช้เงินและแนะนำแนวทางการออมแบบส่วนบุคคล
2. วัตถุประสงค์
    1. เพื่อให้ผู้ใช้งานสามารถบันทึกรายรับ-รายจ่ายได้อย่างง่ายดายผ่านแชท LINE
    2. เพื่อจัดเก็บข้อมูลและแสดงผลผ่านระบบอัตโนมัติด้วย n8n และฐานข้อมูล (Google Sheets)
    3. เพื่อใช้ AI วิเคราะห์พฤติกรรมการใช้เงินรายเดือน และแนะนำแนวทางการออมอย่างเหมาะสม
    4. เพื่อสร้างความตระหนักรู้ด้านการเงินส่วนบุคคลและส่งเสริมวินัยทางการเงิน
3. ขอบเขตของระบบ (Scope)
    • รับข้อความจากผู้ใช้ผ่าน LINE เช่น "รายจ่าย 120 ข้าวเย็น"
    • บันทึกข้อมูลลงใน Google Sheet
    • AI วิเคราะห์รายจ่ายเทียบกับแผนที่ตั้งไว้ เช่น “ใช้เกิน 85% แล้ว”
    • ตอบกลับผ่าน LINE เป็นข้อความหรือ Flex Message
    • รองรับการสรุปยอดรายวัน รายเดือน และสร้างแผนการใช้เงินล่วงหน้า
4. เทคโนโลยีที่ใช้ (Tools & Platform)
รายการ	รายละเอียด
 LINE Messaging API	รับข้อความจากผู้ใช้
 n8n	ระบบ Workflow Automation เชื่อมต่อ LINE, AI, Google Sheets
 OpenAI GPT / Gemini	วิเคราะห์ข้อมูลและแนะนำแนวทางการเงิน
 Google Sheets / MySQL	จัดเก็บข้อมูลรายรับรายจ่าย
 Webhook / HTTP API	ส่งข้อมูลข้ามระบบ

5. ประโยชน์ที่คาดว่าจะได้รับ
    • ผู้ใช้งานสามารถจัดการการเงินของตนเองได้ง่ายขึ้น
    • ลดความเสี่ยงจากการใช้จ่ายเกินตัว
    • ได้รับคำแนะนำทางการเงินจาก AI แบบส่วนบุคคล
    • สามารถนำไปต่อยอดเป็นระบบวางแผนการเงินครอบครัวหรือธุรกิจขนาดเล็ก
